# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-29

### Added
- Initial release of Geyser Equity Research Report Generator
- Comprehensive financial analysis with 40+ metrics
- Automated peer comparison and benchmarking
- Sentiment analysis from analyst recommendations
- Quantitative scoring system with weighted categories
- Professional markdown report generation
- Interactive visualizations with Plotly
- Environment-based configuration (.env support)
- YAML configuration file support
- Centralized logging with colored console output
- Disk-based caching with TTL support
- Input validation and sanitization
- Comprehensive test suite with pytest
- CI/CD pipeline with GitHub Actions
- Pre-commit hooks for code quality
- Type hints and mypy support
- CLI enhancements with argparse
- Progress bars for long operations
- Parallel peer data fetching
- Docker support (planned)
- API wrapper (planned)

### Features
- **Data Collection**: Fetch data from Yahoo Finance
- **Financial Analysis**: Valuation, profitability, growth, health metrics
- **Peer Comparison**: Industry peer identification and comparison
- **Sentiment Analysis**: Analyst recommendations and earnings history
- **Scoring Engine**: Weighted investment scoring (1-10 scale)
- **Report Generation**: Markdown reports with bull/bear cases
- **Visualizations**: Price charts, trends, peer comparisons

### Technical Improvements
- Modular architecture with clear separation of concerns
- Comprehensive error handling and logging
- Caching layer for improved performance
- Configuration management system
- Input validation and security enhancements
- 80%+ test coverage target
- Code quality enforcement (black, flake8, isort, mypy)
- Security scanning (safety, bandit)

### Documentation
- Comprehensive README with examples
- Architecture documentation
- Contributing guidelines
- API documentation (planned)
- Code examples and tutorials
